# jmcunst-library

A personal reading log — books I own, reviews, notes, and things worth remembering.

## Directory Structure

```
jmcunst-library/
├── books/
│   ├── tech/
│   │   └── evolutionary-architecture/
│   │       ├── README.md
│   │       └── concepts/
│   ├── essay/
│   ├── novel/
│   ├── self-help/
│   └── others/
├── templates/
│   ├── book-template/
│   └── concept-template.md
└── reading-list.md
```

## How to Use

1. 책을 읽기 시작하면 해당 카테고리에 책 폴더 생성
   ```bash
   cp -r templates/book-template books/카테고리/책이름
   ```
2. `README.md`에 책 요약 작성
3. `concepts/` 폴더에 배운 개념들을 개별 파일로 정리
